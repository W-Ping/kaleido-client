# Kaleido-client
**一个辅助Coding的桌面应用工具。**

kaleido工具前端源码 [后端源码](https://github.com/wp2code/kaleido)
## Functionality
- Java代码生成和预览

## Project Setup

### Install

```bash
$ yarn install
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
